Sizing

Sizing is mainly done thru REM's. Root font-size is set to 62.5%. All major browers have a default font-size setting of 16px. By setting it to 62.5% we basicly set font-size to 10px. This makes doing REM calculations really easy because 1rem now equals 10px. So whenever you read a value in REM in this article, just multiply it by 10 to get the corresponding value in pixels.

Containers

We have three sizes for containers: small, medium and large.

Size Value CSS class
small 55rem container-small
medium 55rem container-medium
large 100rem container-large

Typography

As mentioned above, the font size is set on the html element, we also set font size on body to 1.6rem. All font sizing cascades from here.

All bottom margins are set to .5em, which translates to half the font size of the heading itself.

Buttons

button elements, input's with submit and button types and a.button all render the same.

There are three different styles for buttons: regular, inverted and muted.

Buttons can also be full width.
