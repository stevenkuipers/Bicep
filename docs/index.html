Quick start

Use CSS

To use the CSS file as provided, add a link to it in the head of your HTML page.

Use Sass

  1. Clone the project
    • git clone https://github.com/stevenkuipers/Bicep.git
  2. Install dependencies
    • npm install
  3. Edit variables to your need
  4. build css file
    • npm run build-css

Sizing

Sizing is mainly done thru REM's. Root font-size is set to 62.5%. All major browers have a default font-size setting of 16px. By setting it to 62.5% we basicly set font-size to 10px. This makes doing REM calculations really easy because 1rem equals 10px. So whenever you read a value in REM in this article, just multiply it by 10 to get the corresponding value in pixels.

Containers

There are three sizes for containers: small, medium and large.

Size Value CSS class
small 55rem container-small
medium 75rem container-medium
large 100rem container-large

Font size

As mentioned above, the font size is set on the html element, we also set font size on body to 1.6rem. All font sizing cascades from here.

Typography

All bottom margins are set to .5em, which translates to half the font size of the heading itself.

Buttons

button elements, input's with submit and button types and a.button all render the same.

Buttons can also be full width.

const randomIntRange = (min, max) => {
  min = Math.ceil(min);
  max = Math.floor(max);
  return Math.floor(Math.random() * (max - min)) + min;
}
